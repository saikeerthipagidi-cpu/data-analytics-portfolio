# Business Intelligence Dashboard

## Project Overview
The Business Intelligence Dashboard converts raw retail data into meaningful insights to help stakeholders monitor business performance. The dashboard focuses on **topline performance, regional health, and brand-level profitability**, enabling users to track key performance indicators (KPIs) against predefined targets.

The report highlights monthly performance trends, product profitability, and geographic distribution of transactions to support data-driven decision-making.


## Key Features & Visualizations

### Executive KPI Summary
High-level performance indicators displaying:
- Current Month Transactions
- Current Month Profit
- Current Month Returns

Each KPI includes **goal comparison indicators** with color-coded status:
- Green → Performance exceeds target
- Red → Performance below target


### Topline Performance Analysis
A detailed brand-level table showing:
- Total Transactions
- Total Profit
- Profit Margin
- Return Rates

Conditional formatting (data bars and heatmaps) is used to quickly highlight performance differences across brands.

Profit margins range approximately **between 58% and 63%** across brands.


### Geographic Performance Analysis
A **Treemap visualization** displays transaction distribution across regions:
- USA
- Mexico
- Canada

Interactive slicers allow users to filter the report dynamically by region for deeper analysis.


### Temporary Revenue Trends
A **Weekly Revenue Trend Chart** illustrates sales fluctuations across the year **1998**, helping identify seasonality patterns and weekly revenue behavior.



### Revenue vs Target Tracking
A **radial gauge visualization** compares:

- Current Revenue: **$1M**
- Annual Target: **$2M**

This provides a quick snapshot of progress toward the annual revenue goal.

## Key Insights

**Market Dominance**
- The **USA contributes the largest share of transactions**, followed by Mexico.

**Brand Performance**
- **Hermanos** leads in total transactions.
- **Plato** demonstrates the **highest profit margin (63.55%)**.

**Operational Alert**
- Current month **returns increased by 2.9% above the target**, suggesting potential issues in product quality or return policies.
